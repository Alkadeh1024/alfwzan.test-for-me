يمكنك تصدير البيانات كملف site.json للنسخ الاحتياطي، أو استيراد ملف موجود.

إعدادات النشر إلى GitHub

هذه البيانات تُحفظ على جهازك فقط (في المتصفح) وتُستخدم لرفع تعديلاتك إلى مستودع الموقع على GitHub.

لإنشاء التوكن: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → أعطه صلاحية «Contents: Read and write» على مستودع الموقع فقط.
